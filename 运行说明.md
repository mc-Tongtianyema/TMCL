# TMCL启动器运行说明

## 🚀 快速运行

### 方法一：使用稳定版启动脚本（推荐）

```bash
python start_tmcl.py
```

这个脚本是我们专门优化的稳定版本，能够：
- ✅ 正确初始化所有核心组件
- ✅ 提供详细的启动日志和状态反馈
- ✅ 优雅处理各种异常情况
- ✅ 即使UI有兼容性问题也能正常运行

### 方法二：使用测试脚本验证功能

```bash
python simple_test.py
```

这个脚本用于测试核心功能是否正常工作，会提供直观的测试结果。

## 💻 运行环境要求

- Python 3.8或更高版本
- 已安装PyQt5库
- 已安装所有依赖包（可以通过requirements.txt安装）

## 📝 运行注意事项

1. **首次运行**：
   - 系统会自动创建配置文件和日志目录
   - 配置文件位于：`C:\Users\[用户名]\AppData\Roaming\TMCL\config.json`
   - 日志文件位于：`C:\Users\[用户名]\AppData\Roaming\TMCL\logs\`

2. **关于UI界面**：
   - 目前UI组件存在一些主题相关的兼容性问题
   - 核心功能（配置管理、版本管理、游戏启动等）全部正常工作
   - 启动脚本会自动处理这些问题并给出相应提示

3. **错误排查**：
   - 如果运行出现问题，请查看日志文件获取详细信息
   - 可以先运行`simple_test.py`验证基础功能是否正常

## 🎯 核心功能状态

✅ **已确认工作正常的功能**：
- 配置管理器初始化和配置加载
- 日志系统记录
- BMCL API客户端连接
- 版本管理器初始化
- 游戏启动器功能
- PyQt5基础界面功能

⚠️ **需要注意的问题**：
- UI主题属性兼容性问题（不影响核心功能）

祝您使用愉快！如果有任何问题，请检查日志文件或重新运行测试脚本。
